CIE Colour Match API

Endpoints

CIE LMS Cone Fundamentals with optional logarithmic values & 9 significant values:

  • Path: '/lms/{route}', where 'route' has to either be:
    • /calculation: A map containing direct computational results and plot values.
      • Method: GET
      • Content-Type: application/json
    • /sidemenu: A HTML page containing a description about the function.
      • Method: GET
      • Content-Type: text/html
    • /plot: A HTML page containing an interactable plot of the function.
      • Method: GET
      • Content-Type: text/html

  • Parameters (shared between both routes above):
    • 'field_size': Mandatory, wished field_size between 1.0-20.0, float type.
    • 'age': Mandatory, wished age between 20-80, float type.
    • 'min': Optional, minimum value for domain between 390.0-400.0 nm (default = 390.0), float type.
    • 'max': Optional, maximum value for domain between 700.0-830.0 nm (default = 830.0), float type.
    • 'step_size': Optional, value for step_size between 0.1-5.0 nm (default = 1.0), float type.
  • Additional parameters (placed as comma-separated list value in "optional" parameter (such as 'optional=log,base,info,norm'):
    • 'log': Activates logarithmic values for function, use by placing log into optional.
    • 'base': Activates 9 significant figures for function, use by placing base into optional.
  • Examples:
    • /lms/calculation?field_size=1.0&age=64&min=390.0&max=830.0&step_size=0.5&optional=log,base
    • /lms/calculation?field_size=2.0&age=23
    • /lms/sidemenu?field_size=2.0&age=23&max=700&min=400
    • /lms/plot?field_size=2.0&age=23&max=700&min=400

MacLeod-Boynton ls chromaticity diagram:

  • Path: '/lms-mb/{route}', where 'route' has to either be:
    • /calculation: A map containing direct computational results and plot values.
      • Method: GET
      • Content-Type: application/json
    • /sidemenu: A HTML page containing a description about the function.
      • Method: GET
      • Content-Type: text/html
    • /plot: A HTML page containing an interactable plot of the function.
      • Method: GET
      • Content-Type: text/html

  • Parameters (shared between both routes above):
    • 'field_size': Mandatory, wished field_size between 1.0-20.0, float type.
    • 'age': Mandatory, wished age between 20-80, float type.
    • 'min': Optional, minimum value for domain between 390.0-400.0 nm (default = 390.0), float type.
    • 'max': Optional, maximum value for domain between 700.0-830.0 nm (default = 830.0), float type.
    • 'step_size': Optional, value for step_size between 0.1-5.0 nm (default = 1.0), float type.
  • Additional parameters (placed as comma-separated list value in "optional" parameter (such as 'optional=log,base,info,norm'):
    • 'info': Returns instead sidemenu information containing the elements below, use by placing info into optional.
      • 'norm': Normalization coefficients.
      • 'white': Chromaticity point of Illuminant E.
      • 'tg_purple': Tangent points of purple line.
  • Examples:
    • /lms-mb/calculation?field_size=3.0&age=32&min=390.0&max=830.0&step_size=0.75
    • /lms-mb/calculation?field_size=5&age=21&optional=info
    • /lms-mb/sidemenu?field_size=1.5&age=51&max=700
    • /lms-mb/plot?field_size=1.5&age=51&max=700
